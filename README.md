# LUXE Skincare

A luxury skincare e-commerce platform built to demonstrate modern web development practices and production-ready deployment workflows.

**Live Demo**: [https://shahzada-shah.github.io/luxe-skincare/](https://shahzada-shah.github.io/luxe-skincare/)

---

## Tech Stack

- **React 18** - UI framework with functional components and hooks
- **TypeScript** - Type-safe development
- **Vite** - Fast build tool and dev server
- **Tailwind CSS** - Utility-first styling
- **React Router v6** - Client-side routing
- **GitHub Actions** - Automated CI/CD pipeline
- **GitHub Pages** - Production hosting

---

## Purpose

This project showcases:
- Building a full-featured e-commerce SPA from scratch
- Implementing clean, maintainable code architecture
- Creating elegant, responsive UI/UX with smooth animations
- Managing complex state (shopping cart, wishlist, authentication)
- Setting up automated deployment pipelines
- Handling production-ready routing and asset optimization

---

## Key Features

✅ **Product Management** - Dynamic catalog with filtering, search, and detail pages  
✅ **Shopping Experience** - Cart and wishlist with localStorage persistence  
✅ **Authentication Flow** - Login/signup pages with password validation  
✅ **Checkout Process** - Multi-step checkout with form validation  
✅ **Responsive Design** - Mobile-first, works on all screen sizes  
✅ **CI/CD Pipeline** - Automatic builds and deployment on push  
✅ **Custom 404 Page** - Branded error handling

---

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```
